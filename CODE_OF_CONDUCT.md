# Code of Conduct

This project adheres to the [Contributor Covenant](https://www.contributor-covenant.org/) Code of Conduct, which establishes expectations for participation in our community to ensure a welcoming and harassment-free experience for everyone.

As a member of the .NET Foundation ecosystem, this project also follows the [.NET Foundation Code of Conduct](https://dotnetfoundation.org/code-of-conduct).

## Reporting Issues

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by opening an issue in this repository or contacting the project maintainers directly. All complaints will be reviewed and investigated promptly and fairly.
